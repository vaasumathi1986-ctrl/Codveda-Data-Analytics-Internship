# Data Analytics Internship - Codveda

This repository contains the projects and tasks completed during my Data Analytics internship track with Codveda. All analysis pipelines and visualizations were developed using **Python** within a **VS Code** environment.

## 📂 Project Structure

* **`codveda_L1_T1`**: Task 1 - Data Cleaning & Preprocessing (Handling missing values, duplicate removal, and data integrity checks on Churn dataset).
* **`codveda_L1_T2_EDA`**: Task 2 - Exploratory Data Analysis (Statistical summaries and trend visualizations like Histograms, Box plots, and Scatter plots on House Prediction dataset).
