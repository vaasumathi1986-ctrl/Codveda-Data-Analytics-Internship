# Level 2 - Task 3: K-Means Clustering

## Objective
Apply K-Means Clustering on the Iris dataset to group similar flowers based on their features.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps Performed
1. Loaded the Iris dataset.
2. Selected numerical features for clustering.
3. Standardized the data using StandardScaler.
4. Used the Elbow Method to determine the optimal number of clusters.
5. Applied K-Means Clustering with 3 clusters.
6. Generated cluster labels.
7. Visualized the clusters using a scatter plot.

## Files
- 1) iris.csv
- task3_kmeans.py
- screenshots/

## Output
- Dataset preview
- Standardized data
- Elbow Method graph
- Cluster labels
- K-Means cluster visualization

## Result
Successfully implemented K-Means Clustering on the Iris dataset and identified 3 optimal clusters using the Elbow Method.
