# Level 2 - Task 1: Regression Analysis

## Objective
Build a Linear Regression model to predict house prices using housing-related features.

## Tools Used
- Python
- Pandas
- Scikit-learn

## Steps Performed
1. Loaded the dataset.
2. Selected features and target variable (MEDV).
3. Split data into training and testing sets.
4. Trained a Linear Regression model.
5. Predicted house prices.
6. Evaluated the model using MSE and R² Score.
7. Analyzed feature coefficients.
