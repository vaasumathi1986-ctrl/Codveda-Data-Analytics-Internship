# Level 3 - Task 3: Sentiment Analysis

## 📌 Objective

Perform Sentiment Analysis on textual data by preprocessing text, analyzing sentiment distribution, and visualizing the most frequent words using Python.

---

## 📂 Dataset

**Dataset:** `3) Sentiment dataset.csv`

The dataset contains customer reviews/social media text along with their corresponding sentiment labels.

Main columns used:
- **Text** – Review or comment text
- **Sentiment** – Sentiment label (Positive, Negative, Neutral)

---

## ⚙️ Project Workflow

### 1. Data Loading
- Loaded the dataset using Pandas.
- Displayed dataset shape and previewed the first five records.

### 2. Text Preprocessing
- Converted text to lowercase.
- Removed URLs.
- Removed punctuation and special characters.
- Removed English stopwords using NLTK.
- Created a new **Cleaned_Text** column.

### 3. Sentiment Analysis
- Counted the number of reviews in each sentiment category.

### 4. Data Visualization
- Created a **Sentiment Distribution Bar Chart** using Seaborn.
- Generated a **Word Cloud** to visualize the most frequently occurring words.
