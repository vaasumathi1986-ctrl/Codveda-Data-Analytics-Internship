# Level 3 - Task 2: Interactive Dashboard using Power BI

## 📌 Objective

Develop an interactive Power BI dashboard to analyze customer churn data and present meaningful business insights using different visualizations.

---

## 📂 Dataset

**Dataset:** `churn-bigml-80.csv`

The dataset contains customer information such as:

- Customer Churn
- Account Length
- Customer Service Calls
- Total Day Minutes
- Total Day Calls
- International Plan
- Voice Mail Plan
- State
- Charges and Usage Details

---

## 📊 Dashboard Visualizations

The dashboard includes the following visualizations:

- 📊 Bar Chart – Customer Churn Distribution
- 🥧 Pie Chart – Churn Percentage
- 📈 Line Chart – Average Total Day Minutes by Customer Service Calls
- 🗺️ Treemap – Customer Distribution by State
- 🔢 Card – Total Customers

---

## 🛠️ Tools Used

- Microsoft Power BI Desktop
- CSV Dataset

---

## 📷 Screenshots

The dashboard screenshots are available in the **Screenshots** folder.

---

## 📈 Key Insights

- Most customers have **not churned**.
- Churned customers represent a smaller percentage of the dataset.
- Customer service calls show variations in average day-minute usage.
- Customer distribution differs across states.
